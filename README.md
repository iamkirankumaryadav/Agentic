# Agentic AI 🤖

### AI Agents
```
Definition:
A smart system that can understand, think, plan, decide, and take actions to complete a task.

Type of Agents:
1. Reactive Agent: Just responds
2. Planning Agent: Think step-by-step
3. Autonomous Agent: Works with minimal human input 
```

### Agentic AI
```
Definition:
1. Type of AI that uses AI agents to get work done autonomously.
2. Agentic AI can act like an independent agent (It can plan, decide, and take actions to complete a task)
3. AI that can think, plan, act, and improve to complete tasks like human assistant agents.

Real Life Analogy:
👉🏻 Think of a personal agent:
You say: "Book a trip to Goa"
1. Searches Flights
2. Compares Prices
3. Book Tickets
4. Send Confirmation

Core Components:
1. Thinking (LLM Models): Understands the requests/tasks and decides what to do.
2. Planning: Break tasks into steps 👉🏻 Search → Compare → Book
3. Tool Usage: APIs, Databases, Browsers, etc.
4. Reflection (Self Check): Reviews its own work, Fixes mistakes

Applications:
1. Dev Assistants (Auto-Debugging Code)
2. Customer Support Automation
3. Data Analysis Agents
4. Workflow Automation (Operations, Ticketing)
```

### Reflection
```
What is Reflection?
1. The AI looks at its own work and asks "Did I do this correctly? Can I improve it?".
2. The AI self reviews and improves its own output before generating final response.
3. Pros: Better accuracy, few mistakes, more complete and relevant response.

Application:
1. Code Generation (Fixing Bugs)
2. Chatbots (Improving Responses)
3. Research Agents (Fact Checking)
4. Autonomous Systems (Decision Validation) 

How Reflection works in Agentic AI❓
- An AI agent doesn’t just answer once. It follows a loop:
1. Generate answer
2. Check its own answer
3. Find mistakes or gaps
4. Improve the answer
5. Repeat (if needed)
👉🏻 Think → Self Review → Improve → Finally Generate
```
